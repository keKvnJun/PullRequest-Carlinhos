## Sobre - Lugares que Queremos Conhecer

Este projeto é uma página web simples que apresenta perfis de integrantes de um grupo, onde cada integrante compartilha informações sobre um lugar que gostaria de conhecer. O objetivo é treinar reviews em pull requests.

<br>

## Estrutura do Projeto
```
├── index.html             # Página inicial com a lista de perfis 
├── giMed.html             # Página do perfil de Giovanna Medeiros 
├── ricardo.html           # Página do perfil de Ricardo Cirilo 
├── dias.html              # Página do perfil de Davi Dias 
├── tutu.html              # Página do perfil de Arthur do Vale 
├── murilinho.html         # Página do perfil de Murilo Rosa 
├── kevin.html             # Página do perfil de Kevin Jun 
├── style.css              # Estilo principal das páginas 
├── perfil.css             # Estilo específico para a página inicial 
├── img/                   # Diretório com imagens dos perfis 
└── LICENSE                # Licença do projeto (MIT)
```

<br>

## Funcionalidades

- **Página inicial**: Exibe os perfis dos integrantes do grupo com imagens e nomes.
- **Páginas individuais**: Cada integrante tem uma página dedicada com informações sobre o lugar que deseja conhecer, incluindo curiosidades.
- **Design responsivo**: As páginas são adaptadas para diferentes tamanhos de tela.

<br>

## Tecnologias Utilizadas

- **HTML5**: Estrutura das páginas.
- **CSS3**: Estilização e efeitos visuais.
